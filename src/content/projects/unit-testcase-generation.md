A generative AI tool for automating unit test creation. Paste a code snippet or upload a file, and the app produces both the test case scenarios and the actual test code in the same language.

Built with React on the frontend and ASP.NET Core Web API on the backend, using OpenAI's GPT-3.5-turbo via the ChatCompleteAsync API.
