## About This Project

Personal website and blog for Manu Dubey, hosted at manudubey.in via GitHub Pages.
A clean, minimal, fully static site focused on readability and simplicity.

**Tech Stack:**
- Astro (static site generator)
- Markdown for blog content
- GitHub Pages + GitHub Actions for deployment

## Who I Am

Manu Dubey - building a personal website and blog. Learning-oriented, prefers simple
and clear explanations. Values getting things working end-to-end over polish.

## My Preferences

- Minimal dependencies - only add what's truly needed
- Simple solutions over complex abstractions
- Mobile-friendly by default
- Clean typography and spacing over flashy design
- No heavy UI frameworks unless necessary
- Personal, honest writing tone - not corporate
